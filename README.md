## Installation

```bash
npm install
```

### iOS

```bash
cd ios && pod install
npx react-native run-ios
```

### Android

```bash
npx react-native run-android
```

## Build for Production

### iOS
```bash
cd ios
xcodebuild -workspace BlackRoad.xcworkspace -scheme BlackRoad archive
```

### Android
```bash
cd android
./gradlew assembleRelease
```
